# Портфолио — демо-лендинги

Публичный репозиторий с дизайн-макетами для портфолио. Каждый проект — самодостаточный статический `index.html` без сборки и внешних зависимостей.

- **[/coretech/](./coretech/)** — CoreTech: экосистемы для сетей медицинских клиник (сайт, приложение, интеграция с МИС).
- **[/coffeesmart/](./coffeesmart/)** — CoffeeSmart: сеть умных кофеен самообслуживания, маркетинговая стратегия 2026–2027.

Корневой `index.html` — общая hub-страница со ссылками на оба проекта.

## Деплой

GitHub Pages, воркфлоу `.github/workflows/deploy.yml`: публикует весь репозиторий целиком при каждом пуше в `main`.

Живая версия: https://kokojamdo384.github.io/landing/
