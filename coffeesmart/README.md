# CoffeeSmart — Landing Page

Одностраничный скролл-лендинг для маркетинговой стратегии сети умных кофе-точек самообслуживания «CoffeeSmart» (2026–2027). Дизайн-макет для портфолио: тёмная тема с зелёно-голубым природным акцентом, имитация 3D-объектов через чистый CSS, скролл-анимации и адаптивная вёрстка.

Технологии: чистый HTML/CSS/JS, без сборщиков и зависимостей — весь сайт умещается в один файл `index.html`.

## Структура

- `index.html` — вся страница (разметка, стили, скрипты)
- `.nojekyll` — отключает обработку Jekyll на GitHub Pages

## Локальный просмотр

Просто откройте `index.html` в браузере — сборка не требуется.

## Деплой

Сайт статический и не требует сервера. Задеплоен через GitHub Pages из этого репозитория.
