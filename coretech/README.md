# CoreTech — экосистемы для медицинских сетей

Демо-лендинг для портфолио: одностраничный сайт для вымышленного IT-агентства
CoreTech, продающего разработку сайтов, приложений и интеграций с МИС для сетей
медицинских клиник.

Самодостаточный `index.html` — без сборки, без внешних зависимостей (шрифты,
скрипты и стили встроены). Открывается напрямую в браузере или деплоится как
статика.

## Деплой

Часть общего портфолио-репозитория. Публикуется на GitHub Pages воркфлоу
`.github/workflows/deploy.yml` при каждом пуше в `main`.

Живая версия: https://kokojamdo384.github.io/landing/coretech/
